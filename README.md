## ğŸ“ Struktur Folder

Workspace ini menggunakan struktur standar Java di VS Code:

-   `src`: **(PENTING)** Folder utama tempat semua source code (`.java`) disimpan.
-   `lib`: Folder buat nyimpen library tambahan/dependency (biasanya kosong kalau gak pake lib luar).
-   `bin`: Folder hasil compile (`.class`). Isinya digenerate otomatis, jadi gak usah diotak-atik.

> **Tips:** Kalau mau nambahin file baru, pastikan taruh di dalam folder `src`.
